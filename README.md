# Abdulmajeed Munassar - Professional Portfolio Website

A highly professional, modern, and **bilingual (Arabic & English)** portfolio website designed to showcase Abdulmajeed Munassar's expertise in Information Technology, Software Development, and Database Management.

## Features

- **Bilingual Support**: Easily switch between Arabic and English content.
- **Professional Design**: Clean, modern, and responsive layout.
- **Interactive Elements**: Smooth scrolling, animations, and dynamic content loading.
- **Sections Included**: Home, About Me, Technical Skills, Professional Experience, Education & Certifications, and Contact.

## File Structure

```
portfolio/
├── index.html       # Main HTML file with bilingual structure
├── styles.css       # CSS styling, including RTL support for Arabic
├── script.js        # JavaScript for interactivity and animations
├── content.js       # JavaScript for handling bilingual content and switching
├── content.json     # JSON file containing all bilingual text content
├── profile.jpg      # Profile photograph
└── README.md        # This file
```

## How to Use

### 1. Upload to GitHub
1. **Create a new repository** on GitHub (e.g., `abdulmajeed-portfolio`).
2. **Upload all files** from this folder (`index.html`, `styles.css`, `script.js`, `content.js`, `content.json`, `profile.jpg`, `README.md`) to the root of your new repository.

### 2. Activate GitHub Pages
1. Go to your repository **Settings** → **Pages**.
2. Under **Source**, select the branch **`main`** (or `master`) and the folder **`/(root)`**.
3. Click **Save**.
4. Your website will be live in a few minutes at the URL provided by GitHub Pages (e.g., `https://yourusername.github.io/abdulmajeed-portfolio`).

### Customization

- **Update Content**: All text content is managed in the `content.json` file. Edit this file to update your information in both English (`"en"`) and Arabic (`"ar"`).
- **Replace Image**: Replace `profile.jpg` with your own image.
- **Change Colors**: Edit `styles.css` and modify the CSS variables at the top (e.g., `--primary-color`, `--accent-color`).
